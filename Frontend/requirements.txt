Authentication:
    The user should be able to create an account,
    get verified, 
    log in to the account and
    reset their password as well.
Tasks:
    The user should be able to create tasks which could be based on some category or not.
    The user should be able to create categories based on their preference of structure for their tasks.
    The user should be able to update a particular task of their choice.
    The user should be able to mark a task as completed.
    The user should be able to delete tasks.
    The user should be able to delete categories.
Profile: 
    The user should be able to upload a profile picture for their account.
Interface:
    The user should be able to see all their tasks on the Dashboard.
    The User should be able to view tasks based on categories if such categories are available.
